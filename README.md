# CelebA
## Foreground_separation1
Image matting on CelebA dataset images
The data set is availabe at: http://mmlab.ie.cuhk.edu.hk/projects/CelebA.html
The Foreground extraction is done on a single image from the dataset based on colour features.
Prequisites:
Install the following packages:-
* numpy
* cv2
* matplotlib
* skimage
* scipy
